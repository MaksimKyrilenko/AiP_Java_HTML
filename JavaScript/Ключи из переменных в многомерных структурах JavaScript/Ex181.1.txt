№1

Для решения этой задачи нужно использовать значения переменных lang и month для получения нужного месяца из объекта months. Для этого можно использовать следующий код:

console.log(months[lang][month]);

Например, если lang равен 'ru', а month равен 5, то на экран будет выведено 'июнь'.

Для решения этой задачи нужно использовать значения переменных year, month и day для получения нужного дела из объекта affairs. Для этого можно использовать следующий код:

console.log(affairs[year][month][day]);

Например, если year равен '2018', month равен 11, а day равен 29, то на экран будет выведен массив ['дело111', 'дело112', 'дело113'].
