1. В консоль будет выведено число 1, так как при вызове функции func(num) происходит передача значения переменной num в качестве аргумента, но внутри функции значение num перезаписывается на 2, но это не влияет на значение переменной num вне функции.
2. В консоль будет выведено число 2, так как внутри функции func() переменная num не объявлена с использованием ключевого слова let или var, поэтому она становится глобальной и перезаписывается на 2. При выводе в консоль переменной num после вызова функции будет выведено последнее ее значение - 2.
3. В консоль будет выведено число 1, так как переменная num внутри функции func() объявлена с использованием ключевого слова let, что создает локальную переменную с тем же именем, что и глобальная переменная num. Перезапись значения локальной переменной не влияет на значение глобальной переменной. Поэтому при выводе в консоль переменной num после вызова функции будет выведено ее глобальное значение - 1.
