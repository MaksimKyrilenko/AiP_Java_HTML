№1: Ошибка заключается в том, что автор не создал внутренний массив для каждого элемента внешнего массива. Необходимо добавить строку arr[i] = []; перед вложенным циклом.

Исправленный код:

let arr = [];

for (let i = 0; i < 3; i++) {
  arr[i] = []; 
  
  for (let j = 1; j <= 5; j++) {
    arr[i].push(j);
  }
}

console.log(arr);
