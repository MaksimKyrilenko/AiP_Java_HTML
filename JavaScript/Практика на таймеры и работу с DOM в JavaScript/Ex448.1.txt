Функция Number используется для преобразования значения атрибута value в число, так как изначально значение атрибута хранится в виде строки. Если не использовать функцию Number, то произойдет конкатенация строк, а не сложение чисел. То есть, например, при значении value = "1" через одну секунду значение станет равным "11", а не 2.
