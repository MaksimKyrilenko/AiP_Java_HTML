№1
1. Создается массив tds, содержащий все ячейки таблицы.
2. Устанавливается переменная color со значением 'color1', которая будет использоваться для определения текущего цвета.
3. Для каждой ячейки таблицы устанавливается обработчик события click, который выполняет следующие действия:
   - Проверяет текущий цвет (color).
   - Если цвет равен 'color1', то он изменяется на 'color2', и наоборот.
   - Добавляется класс текущего цвета к ячейке.
№2
1. Создается массив tds, содержащий все ячейки таблицы.
2. Устанавливаются переменные i и colors.
3. Для каждой ячейки таблицы устанавливается обработчик события click, который выполняет следующие действия:
   - Добавляется класс из массива colors с индексом i к ячейке.
   - Индекс i увеличивается на 1.
   - Если i достигает значения colors.length, то он сбрасывается обратно на 0.
