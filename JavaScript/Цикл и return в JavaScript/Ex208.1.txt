В консоль выведется значение 1. Функция должна вычислять сумму чисел от 1 до переданного аргумента num, но из-за неправильного размещения оператора return внутри цикла for, функция сразу же возвращает сумму после первой итерации (когда i равно 1). 

Автор хотел вычислить сумму чисел от 1 до переданного аргумента num.

Исправленный код:

function func(num) {
  let sum = 0;
  
  for (let i = 1; i <= num; i++) {
    sum += i;
  }
  
  return sum;
}

console.log( func(5) ); // выведет 15
